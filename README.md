`Custom-UI` 是一款基于 `Vue` ，在学习```element-ui``` 源码的闲暇之余，**吃饱了撑着** 对element-ui库进行二次封装和尝试组件改造和新创，方便之后项目中能够更好的使用和提高效率。

## 由来

- 公司项目需要，毕竟每个公司都应该有自己的组件库
- 为了更好的学习 `element` 
- 主要是想给element-ui二次封装提供一个更好的展示平台
- 本项目参考来源：Hai-UI;
